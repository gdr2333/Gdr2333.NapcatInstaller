### Gdr2333.NapcatInstaller

注意：这不是官方安装器。官方安装器在https://github.com/NapNeko/NapCat-Installer。

本安装器特色：

- 专注于本机安装
- 真正的多系统/架构支持
- 支持自动写入systemd配置文件
- 支持保配置升级

目前支持的系统：

- 完全支持：
- - Debian 12或更高 AMD64/ARM64
- - Fedora 42或更高 AMD64/ARM64
- - AlmaLinux 10 AMD64/ARM64
- 完全支持（未测试）：
- - RockyLinux 8/9 AMD64/ARM64
- - AlmaLinux 8/9 AMD64/ARM64
- - Ubuntu 22.04+ AMD64/ARM64
- 部分支持：
- - RHEL 8/9 AMD64/ARM64
- - OpenCloudOS 8/9 AMD64/ARM64
- - OpenSUSE 15+ AMD64/ARM64
- - SLED SLES 15+ AMD64/ARM64
- - CentOS Stream 8/9 AMD64/ARM64
- - Devuan 5+ AMD64/ARM64
- - // 还有很多。
